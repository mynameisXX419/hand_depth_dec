- 第1轮: +290ms

#### 3. 配对数略有下降
- 第1轮: 41对
- 第2轮: 37对
- 可能原因: 峰值检测阈值或波形特征差异

#### 4. 标准差保持稳定
- 两轮都在 260-270ms 范围
- 说明单次实验内的波动相对稳定
